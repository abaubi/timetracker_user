Els Teus Projectes

⚙️ Connectar amb la teva pròpia base de dades Firebase

Aquesta aplicació s'executa 100% al teu navegador i desa les dades en temps real a la teva pròpia base de dades al núvol (Google Cloud Firestore). D'aquesta manera gaudeixes de privacitat total i control absolut del teu historial, sense intermediaris.

📖 Guia ràpida per configurar Firebase gratis (5 minuts)

Pas 1: Crear el Projecte a Firebase

Ves a la consola oficial: console.firebase.google.com i clica a "Afegeix un projecte". Posa-li un nom (ex: El Meu Tracker) i crea'l (potes desactivar Google Analytics).

Pas 2: Activar Autenticació amb Google

Al menú lateral esquerre de Firebase, entra a Compilació ➔ Authentication i clica Comença.
A la pestanya Sign-in method, tria Google, activa'l, escull el teu correu de suport, i desa.

Pas 3: Crear la Base de Dades Firestore

Al menú lateral, entra a Compilació ➔ Firestore Database i clica Crear base de dades.
Selecciona una ubicació (ex: eur3 o europe-west) i tria el Mode de producció.

Pas 4: Configurar les Regles de Seguretat (MOLT IMPORTANT!)

A la pestanya Regles (Rules) de Firestore, esborra el codi que hi hagi i enganxa exactament això (com que el projecte és teu privat, només tu que t'identifiques amb el teu Google pots escriure-hi):

Clica al botó Publicar.

Pas 5: Obtenir les teves credencials (firebaseConfig)

A dalt a l'esquerra de Firebase, clica l'engranatge ⚙️ Configuració del projecte.
A la pestanya General, baixa fins a "Les teves aplicacions" i clica la icona web </>.
Registra l'aplicació, copia l'objecte firebaseConfig que apareixerà i enganxa'l al quadre de text superior d'aquesta pestanya!

Pas 6: Autoritzar el Domini Web (MOLT IMPORTANT!)

Per seguretat de Google, la finestra d'iniciar sessió es tanca o dóna error si no autoritzes l'adreça d'aquesta web al teu projecte de Firebase.
1. A la consola de Firebase, ves al menú lateral: Compilació ➔ Authentication ➔ Settings (Configuració) ➔ Authorized domains (Dominis autoritzats).
2. Clica Add domain (Afegeix domini) i enganxa l'adreça web actual:
